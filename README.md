# Visualizing UChicago Dissertation Topics (1893–2025)

## Overview

>This project analyzes and visualizes dissertation data from the University of Chicago spanning 1893–2025. Using natural language processing (NLP), network analysis, and GIS techniques, it explores trends in research topics, departmental activity, and interdisciplinary connections over time.

## Dataset

The dataset contains ~32,000 dissertation records with fields including:
- Title
- Abstract / Description
- Keywords
- Department / Division
- Author + ORCID (when available)
- Advisor and committee members

## Data was collected via:
- ProQuest TDM Studio
- UChicago Library dissertation archive

## Goals
- Identify trends in dissertation topics over time
- Analyze departmental and interdisciplinary relationships
- Map spatial and geographic patterns in research
- Explore academic lineage and advisor influence
